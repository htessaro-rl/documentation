title: "Can the # symbol be removed for a KPI with no Low-High interface?"
slug: /faq/remove-hash-symbol
description: ''
tags: [FAQ]
---

If a KPI has no Low High interface it displays **#** symbol.

You can set the Status to Good or anything else you wish. Use Visual KPI Designer to set the Status column, and the drop-down menu to set the value.
